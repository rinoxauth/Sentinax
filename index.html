Simple yet powerful slash commands to manage every aspect of your server security.

/scam fullscan Deep scans channel history to purge existing scam content.
/scam security Provides a real-time security health report of the server.

Common Questions

How does OCR detection work?
Sentinax uses Optical Character Recognition to extract text from images sent in chat. It then compares this text against a vast database of scam keywords and phishing patterns.
Is it free to use?
Yes! Sentinax offers its core security and moderation features for free to help keep the Discord ecosystem safe.
Does it store my messages?
No. Sentinax processes messages in real-time to detect threats and does not store private user data or message history.
